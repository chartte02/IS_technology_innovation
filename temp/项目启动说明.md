# NADS 项目启动说明

## 环境要求

- Python 3.9+（当前 venv: 3.13.5）
- Windows 需安装 Npcap（在线抓包需要，PCAP 回放不需要）

## 快速启动

```bash
# 进入项目目录（venv 已就绪，直接在项目根目录操作）
cd D:\Document\CODE\School\IS_technology_innovation

# 方式 1：GUI 界面
.\venv\Scripts\python.exe main.py

# 方式 2：命令行模式（不需要 GUI）
.\venv\Scripts\python.exe main.py --console

# 方式 3：PCAP 回放（演示推荐，无需管理员权限）
.\venv\Scripts\python.exe main.py --replay tests/test_pcaps/synthetic_attacks.pcap

# 方式 4：运行测试
.\venv\Scripts\python.exe tests/test_signature_match.py -v
```

## PCAP 文件

| 文件 | 说明 |
|------|------|
| `tests/test_pcaps/synthetic_attacks.pcap` | 合成攻击样本（SQLi/XSS/Web/BruteForce） |
| `tests/test_pcaps/http.cap` | HTTP 流量（256KB） |
| `tests/test_pcaps/extended_attacks.pcap` | 扩展攻击样本 |

## 注意

- PCAP 回放模式**不需要管理员权限和 Npcap**，是演示最稳定的方案
- 在线抓包需要以**管理员身份**运行终端
- GUI 包含 5 个 Tab：仪表盘 / 告警列表 / 统计分析 / 特征库管理 / 系统日志
