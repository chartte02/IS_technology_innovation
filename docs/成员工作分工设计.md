**B-1 JA3 指纹提取算法**：
```
1. 识别 TLS Record: 前3字节 = \x16\x03\x01
2. 识别 ClientHello: Handshake Type = \x01
3. 逐字段解析:
   - Protocol Version (2 bytes)
   - Random (32 bytes) → 跳过
   - Session ID (变长) → 跳过
   - Cipher Suites (变长) → 每个2字节，转十进制，-连接
   - Compression Methods (变长) → 跳过非null值
   - Extensions (变长) → 每个2字节 type
4. 拼接: "{Ver}-{Ciphers}-{Extensions}-{Curves}-{Formats}"
5. MD5 → 32字符 JA3 指纹
```

**B-S1 C2 Beaconing 算法**：
```python
# 核心逻辑 (~80行)
class C2BeaconDetector:
    def __init__(self, cv_threshold=0.15, min_samples=5):
        self.connections = {}  # (src, dst, port) → [timestamps]

    def feed(self, src_ip, dst_ip, dst_port, timestamp):
        key = (src_ip, dst_ip, dst_port)
        self.connections.setdefault(key, []).append(timestamp)

    def check(self, key):
        ts = self.connections[key]
        if len(ts) < self.min_samples: return None
        intervals = [ts[i+1]-ts[i] for i in range(len(ts)-1)]
        mean = sum(intervals) / len(intervals)
        if mean == 0: return None
        cv = (sum((x-mean)**2 for x in intervals)/len(intervals))**0.5 / mean
        if cv < 0.05:  return {'severity':'critical', 'cv':cv}
        if cv < 0.15:  return {'severity':'high', 'cv':cv}
        return None
```

```python
# core/ml_anomaly.py (~150行)
from sklearn.ensemble import IsolationForest

class MLAnomalyDetector:
    def __init__(self):
        self.model = IsolationForest(
            n_estimators=100, contamination=0.05, random_state=42
        )
        self.trained = False

    def extract_features(self, host_stats) -> list:
        duration = max(time.time() - host_stats.first_seen, 1)
        return [
            host_stats.conn_count,           # 连接数
            host_stats.syn_count,            # SYN数
            len(host_stats.unique_dst_ports), # 唯一端口数
            len(host_stats.unique_dst_ips),   # 唯一IP数
            host_stats.bytes_sent,            # 发送字节
            host_stats.bytes_received,        # 接收字节
            host_stats.packet_count/duration,  # 包速率
            host_stats.login_failures,         # 登录失败
        ]

    def train(self, all_host_stats):
        X = [self.extract_features(s) for s in all_host_stats.values()]
        if len(X) >= 10:  # 最少需要10个样本
            self.model.fit(X)
            self.trained = True

    def predict(self, host_stats) -> int:
        if not self.trained: return 1
        return self.model.predict([self.extract_features(host_stats)])[0]
        # 返回: -1=异常, 1=正常
```

**C-S1 两阶段检测流程**：
```
Stage 1: IF 粗筛
  活跃主机 → 8维特征 → model.predict() → 异常主机列表

Stage 2: 三源精判（投票制）
  ┌─ MisuseDetector 是否有告警?   → 票数+1
  ├─ BaselineProfile 偏离 > 2σ?   → 票数+1
  └─ ThreatIntel 已知恶意?         → 票数+1

  票数 ≥ 2 → 确认告警(high/critical)
  票数 = 1 → 低优先级(low)
  票数 = 0 → 抑制(误报)
```

```python
    results['pulse_count'] = resp.json().get('pulse_info', {}).get('count', 0)

    results['risk'] = 'HIGH' if results['abuse_score']>50 or results['pulse_count']>5 else 'LOW'
    return results
```
