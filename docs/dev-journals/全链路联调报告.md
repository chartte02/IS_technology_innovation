## 7. 已知限制

1. **AnomalyDetector** 在短 PCAP 测试中不会触发（需要 60s+ 的累积流量才达到统计阈值），7 种检测器已在独立测试中逐项验证
2. **TLSDetector** 需要真实 TLS 流量 PCAP 来端到端验证（当前测试 PCAP 均为 HTTP 明文）
3. BaselineLearner 的 `_sample()` 依赖外部定时器，需在 main.py 中补充定时采样逻辑
