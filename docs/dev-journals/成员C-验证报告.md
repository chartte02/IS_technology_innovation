## 5. 发现与修复

### 修复1："Failed password" 未检测

**位置**: `_check_login_pattern()` L457

**问题**: OpenSSH 最常见登录失败消息 `"Failed password"` 不在检测列表中，6 种模式唯独漏了最常见的一种。

**修复**: 新增一行 `if 'failed password' in payload_str`。

**效果**: SSH 暴力破解检测覆盖率从覆盖 OpenSSH 部分消息 → 覆盖最常用消息。
