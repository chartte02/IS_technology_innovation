## 四、成员B — 数据采集层 + 协议识别

### 4.1 现有代码说明

#### `packet_capture.py`（~250行）

已有的功能：
- `PacketCapture` 类：完整的数据包捕获器
  - `start()` — 使用 Scapy AsyncSniffer 异步抓包（不阻塞）
  - `start_blocking(count)` — 阻塞式抓包（适合简单脚本）
  - `stop()` / `pause()` / `resume()` — 完整的状态控制
  - `replay_pcap(path, speed)` — PCAP 文件回放
  - `add_callback(fn)` / `remove_callback(fn)` — 回调链
  - `get_status()` — 运行时统计（PPS、抓包数、运行时间）
  - `list_interfaces()` / `get_default_interface()` — 网卡枚举

#### `protocol_parser.py`（~350行）

已有的功能：
- `ProtocolParser` 类：完整的协议解析器
  - `parse(packet)` — 支持 Scapy 对象 + 原始 bytes 两种输入

```python
class StreamBuffer:
    def __init__(self):
        self.segments = {}  # {seq_num: bytes}  乱序暂存
        self.next_seq = None  # 期望的下一个序列号
        self.data = bytearray()  # 按序重组后的数据

    def add_segment(self, seq, payload):
        # 1. 如果是重传包（seq < next_seq），忽略
        # 2. 如果 seq == next_seq，直接追加，然后检查是否有缓存的后续包
        # 3. 如果 seq > next_seq，先缓存，等中间包到达
        pass
```

#### 🔴 PDF必做1：ML 识别未知攻击（Isolation Forest）

PDF 原文要求：「机器学习模型识别未知攻击（零日攻击、无文件攻击，弥补特征库短板）」

用 scikit-learn 的 Isolation Forest 替代固定阈值，实现无监督异常检测：

```python
# 新增文件: core/ml_anomaly.py
from sklearn.ensemble import IsolationForest
import numpy as np

class MLAnomalyDetector:
    """
    基于 Isolation Forest 的无监督异常检测器

    特征向量设计（每台主机一个向量）:
    [conn_count, syn_count, unique_ports, unique_ips,
     bytes_sent, bytes_received, packet_rate, login_failures]
    """

    def __init__(self):
        self.model = IsolationForest(
            n_estimators=100,
            contamination=0.05,  # 预期 5% 是异常
            random_state=42
        )
        self._feature_cache = {}  # IP → feature_vector
        self._trained = False

    def extract_features(self, host_stats) -> list:
        """从 HostStats 提取特征向量"""
        duration = max(time.time() - host_stats.first_seen, 1)
        return [
            host_stats.conn_count,
            host_stats.syn_count,
            len(host_stats.unique_dst_ports),
            len(host_stats.unique_dst_ips),
            host_stats.bytes_sent,
            host_stats.bytes_received,
            host_stats.packet_count / duration,  # 包速率
            host_stats.login_failures,
        ]

    def train(self, all_host_stats):
        """用当前所有主机的统计训练模型"""
        X = np.array([self.extract_features(s)
                       for s in all_host_stats.values()])
        self.model.fit(X)
        self._trained = True

    def predict(self, host_stats):
        """返回: 1=正常, -1=异常"""
        if not self._trained:
            return 1
        X = np.array([self.extract_features(host_stats)])
        return self.model.predict(X)[0]
```

答辩时可以说："相比固定阈值方式，Isolation Forest 能自适应不同网络环境的基线，无需手动设定参数。"

```python
def _create_severity_chart(self, stats: dict):
    """根据告警统计数据创建严重度饼图"""
    series = QPieSeries()
    colors = {'critical': QColor('#d32f2f'), 'high': QColor('#f57c00'),

    for sev, count in stats.get('by_severity', {}).items():
        series.append(sev, count)
        if sev in colors:
            series.setColor(len(series)-1, colors[sev])

    chart = QChart()
    chart.addSeries(series)
    chart.setTitle('告警严重度分布')
    chart.setAnimationOptions(QChart.SeriesAnimations)

    chart_view = QChartView(chart)
    return chart_view
```
